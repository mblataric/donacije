Podrška za Donacije

Donacije je iPad aplikacija za izdavanje internih bonova, lokalne izvještaje, backup i ispis na događaju.

Kako zatražiti pomoć

Za prijavu problema ili pitanje otvorite issue na javnom GitHub repozitoriju: github.com/mblataric/donacije/issues.

Osnovne napomene

  • Aplikacija radi lokalno na iPadu i ne zahtijeva stalnu internetsku vezu.
  • Backup se može izvesti kao .backup datoteka.
  • Za direktni ispis potrebno je prethodno spremiti zadani printer u Admin dijelu aplikacije.
